# mg

The MongoDB Launcher.

```
> ./bin/mg.js -h

The MongoDB Launcher
Usage: mg

Examples:
  mg              deploy all unicorns
  mg mongod       only start mongod
  mg mongorest    only start the rest server
```

## dev

### install

```
npm install
```

### test

```
npm test
```

### bake binary

```
npm dist
```
